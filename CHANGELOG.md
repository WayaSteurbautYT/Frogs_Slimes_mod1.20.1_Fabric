# Changelog

All notable changes to Frog & Slime Gamemode will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of Frog & Slime Gamemode
- Custom gamemode system with `/frogslime start` and `/frogslime stop` commands
- Frog and slime helper entities with taming mechanics
- Evolution system through 5 stages (Basic → Advanced → Elite → Master → Final Form)
- Custom particles for evolution, petting, and idle animations
- Dynamic name tags with color-coded evolution stages
- Automatic item collection from killed mobs
- Player transformation system via frog/slime food
- Task & challenge system with 10 unique challenges
- Custom GUI for tasks menu
- Achievement system with 7 custom advancements
- Custom on-screen titles during key moments
- Giant Slime Boss replacing the Ender Dragon
- Unexpected twist ending with Final Evolution
- Custom items:
  - Evolution Stone
  - Final Evolution Crystal
  - Frog Food
  - Slime Food
  - Task Book
  - Dream Sword
  - Technoblade Sword
  - Grian Sword
  - Mumbo Jumbo Sword
  - Mustard Helmet
  - Orphan Shield
  - Prankster Chestplate
- Custom textures for all entities and items
- Spawn eggs for frog and slime helpers

### Changed
- Replaced Ender Dragon with Giant Slime Boss
- Modified end game sequence for twist ending

### Fixed
- Initial bug fixes and stability improvements
